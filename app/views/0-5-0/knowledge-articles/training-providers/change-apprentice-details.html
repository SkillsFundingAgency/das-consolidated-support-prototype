Change apprentice details

The employer must set training provider permissions before you can change apprentice details in the apprenticeship service account.

You can change the apprentice's name, date of birth and training information. The employer must approve any changes

Change an apprenticeship status

You can pause an apprenticeship if you need to:

  • record a break in learning that you’ve agreed with the employer
  • temporarily freeze payments

You can stop an apprenticeship if you need to:

  • withdraw an apprentice from their apprenticeship
  • change the training provider during the course
  • cancel all future payments
! Warning Do not stop a completed apprenticeship.

You must also record the pause or stop on the individualised learner record (ILR).

After the employer makes their final apprenticeship payment, the apprentice will show as ‘completed’ in the apprenticeship service account.

Change an apprentice’s course details

You can change an apprentice’s course details before the course starts. This includes the course name, type of apprenticeship, start date and planned end date.

If you need to change an apprentice’s course name after the course has started, you must:

  • stop the apprenticeship and backdate to the start date
  • update the ILR
  • start a new apprenticeship with the correct details
