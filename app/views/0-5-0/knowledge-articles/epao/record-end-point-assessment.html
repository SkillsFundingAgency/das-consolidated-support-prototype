Record an end-point assessment

You must record all end-point assessment grades in the apprenticeship assessment service. Only record a ‘fail’ if it is defined in the assessment plan.

Select ‘record a grade’ within your account. You’ll need to know the date the apprentice achieved or failed their apprenticeship and their:

  • name
  • unique learner number (ULN)
  • learning option

We check the apprentice’s details using information from the training provider’s individualised learner record (ILR). If you cannot find an apprentice, check that the training provider has their correct details.

If the training provider has incorrect details, they will need to update the ILR before you can request their certificate.
