Set up as a user

Everyone that will use your organisation’s apprenticeship service account must set up their own sign in details. They must not share these details.

You’ll set up your sign in details with your name, email address and a password. You must also accept the terms and conditions.

We’ll send you a verification code by email. You’ll need to enter this to confirm your identity.

If you did not receive a verification code, check:

  • your junk email
  • that you entered the right email address

You can request a new verification code if you entered the wrong email address or did not receive one.
